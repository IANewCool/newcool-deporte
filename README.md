# NewCool Deporte - IND Chile

Portal educativo sobre deporte y vida activa en Chile.

## Demo

[https://deporte.newcool.io](https://deporte.newcool.io)

## Caracteristicas

- Buscador de recintos deportivos
- Programas del Instituto Nacional del Deporte
- Directorio de federaciones deportivas
- Calculadora de frecuencia cardiaca
- Eventos deportivos

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
